Data URI Converter

Data URI Converter is a developer tool with simple UI for Mac, it converts any file to base64 string, especially useful for web developer.

Feature list:

  • Drag & drop;
  • Auto detect file type (extension name ignored);
  • Auto insert "data:mime-type;base64," prefix;
  • Auto copy converted strings to pasteboard;
  • Hide icon from Dock.app;
  • Notification Center supported (require 10.8);
  • History supported (up to 20)。
